# 虚实比对比功能说明

## 功能概述

虚实比分析页面现在自动对比上一期数据,直观展示各指标的变化趋势和百分比,帮助快速识别风险变化。

## 新增展示字段

### 1. 虚实比列

**显示内容**:
```
314.40
↑ 5.95%
```

**说明**:
- 第一行: 当前虚实比
- 第二行: 较上一期变化百分比
- 颜色标识:
  - 🔴 红色↑: 虚实比上升,逼仓风险加剧
  - 🟢 绿色↓: 虚实比下降,逼仓风险缓解

### 2. 仓单量列

**显示内容**:
```
22,769
↑ 24.63%
```

**说明**:
- 第一行: 当前仓单量
- 第二行: 仓单变化百分比
- 颜色标识:
  - 🟢 绿色↑: 仓单增加,可交割库存增加,利空价格
  - 🔴 红色↓: 仓单减少,可交割库存减少,利多价格

### 3. 持仓量列

**显示内容**:
```
408,723
↑ 5.95%
```

**说明**:
- 第一行: 当前持仓量(手)
- 第二行: 持仓变化百分比
- 颜色标识:
  - 🔴 红色↑: 持仓增加,市场需求增加,风险提升
  - 🟢 绿色↓: 持仓减少,市场需求减少,风险降低

## 实际案例分析 (2025-12-05 vs 2025-12-04)

### 案例1: 铁矿石 - 高风险持续恶化 🚨

| 指标 | 当前值 | 变化 | 解读 |
|------|--------|------|------|
| 虚实比 | 314.40 | ↑ 5.95% | 持仓量是仓单的314倍,极高风险 |
| 仓单量 | 1,300 | 持平 | 可交割库存未增加 |
| 持仓量 | 408,723 | ↑ 5.95% | 需求继续增长 |

**结论**:
- 虚实比从296.75上升到314.40
- 持仓量持续增长,但仓单量不变
- 空头交割压力持续加大
- ⚠️ 建议: 高度警惕临近交割月的价格波动

### 案例2: 豆粕 - 中风险逐步加剧 ⚠️

| 指标 | 当前值 | 变化 | 解读 |
|------|--------|------|------|
| 虚实比 | 74.84 | ↑ 3.13% | 从72.57上升 |
| 仓单量 | 23,830 | 持平 | 供应端无变化 |
| 持仓量 | 1,783,520 | ↑ 3.13% | 需求端持续增长 |

**结论**:
- 虚实比缓慢上升,进入中风险区间
- 需要关注仓单变化和交割压力

### 案例3: 豆油 - 风险明显缓解 ✅

| 指标 | 当前值 | 变化 | 解读 |
|------|--------|------|------|
| 虚实比 | 22.73 | ↓ 16.98% | 从27.38大幅下降 |
| 仓单量 | 22,769 | ↑ 24.63% | 可交割库存大幅增加 |
| 持仓量 | 517,582 | ↑ 3.47% | 需求小幅增长 |

**结论**:
- 虚实比下降近17%,风险显著缓解
- 主要原因: 仓单量大幅增加24.63%
- 供应充足,价格压制作用增强
- ✅ 市场相对安全,短期逼仓风险低

## 使用技巧

### 1. 快速识别风险变化

**看虚实比变化百分比颜色**:
- 🔴 连续多日红色↑: 风险持续累积,需要高度关注
- 🟢 绿色↓: 风险缓解,市场趋于健康
- ➡️ 变化<1%: 市场相对稳定

### 2. 分析供需变化

**仓单 vs 持仓的组合**:
- ↑仓单 + ↓持仓 = 供增需减,价格承压
- ↓仓单 + ↑持仓 = 供减需增,风险上升
- ↑仓单 + ↑持仓 = 市场活跃,关注比例
- ↓仓单 + ↓持仓 = 市场萎缩,流动性降低

### 3. 预警信号

**高风险预警条件**:
1. 虚实比 > 100 且连续上升
2. 仓单量持续下降或不变
3. 持仓量持续增长
4. 临近交割月

### 4. 横向对比

- 表格支持按虚实比排序
- 可快速找出风险最高的品种
- 对比同类品种的风险变化趋势
