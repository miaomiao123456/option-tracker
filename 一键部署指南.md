# 🚀 一键部署到公网 - 超简单版

## 只需3步,5分钟完成!

### 第1步: 注册Railway账号 (1分钟)

1. 访问 https://railway.app/
2. 点击右上角 "Login"
3. 选择 "Login with GitHub" (用你的GitHub账号登录)
4. 授权Railway访问你的GitHub

### 第2步: 一键部署 (2分钟)

1. 登录后,点击 "New Project"
2. 选择 "Deploy from GitHub repo"
3. 找到并选择 `miaomiao123456/option-tracker` 仓库
4. 点击 "Deploy Now"

Railway会自动:
- ✅ 检测Python项目
- ✅ 安装依赖
- ✅ 启动服务
- ✅ 分配公网域名

### 第3步: 配置环境变量 (2分钟)

部署开始后:

1. 点击项目名称进入项目详情
2. 点击 "Variables" 标签
3. 点击 "New Variable"
4. 添加以下变量:

```
名称: UQER_TOKEN
值: 190bbd239ab55cb2f3f2919601622b0b793e94c35d7967e0c4b325682eddd981
```

5. 点击 "Add" 保存
6. Railway会自动重新部署

### 完成! 🎉

**等待3-5分钟部署完成后:**

1. 在Railway项目页面,点击 "Settings" 标签
2. 找到 "Domains" 区域
3. 点击 "Generate Domain" 生成公网域名
4. 复制域名,格式类似: `option-tracker-production-xxxx.up.railway.app`

**访问你的网站:**
- 🌐 前端页面: `https://你的域名/frontend`
- 📊 API文档: `https://你的域名/docs`

**示例:**
如果你的域名是 `option-tracker-production-abc123.up.railway.app`
- 前端: https://option-tracker-production-abc123.up.railway.app/frontend
- API: https://option-tracker-production-abc123.up.railway.app/docs

## 🆘 遇到问题?

### 问题1: 部署失败
- 检查GitHub仓库是否公开
- 查看Railway的构建日志

### 问题2: 打开网页显示错误
- 等待3-5分钟让部署完成
- 检查环境变量是否正确配置

### 问题3: 数据不显示
- 刷新页面(Ctrl+F5 或 Cmd+Shift+R)
- 检查浏览器控制台是否有错误
